Call for Proposals

RubyConf KL is intended to be loosely structured and less formal. So we are looking
for talks in many shapes and form, such as presentations, show & tell, lightning talks, workshops, etc.

If you need some inspirations, here are some suggested areas that you can talk about:

  • Agile
  • Architecture
  • Community
  • Cutting Edge
  • Devops
  • Front-end
  • Languages
  • Mobile
  • Tools
  • Open Source
  • Security

Call for proposal ends at 15th June 2016 and selected speakers will be contacted soon after.
